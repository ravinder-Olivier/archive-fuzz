# Archive Fuzz Mk. 1.5
A preset enabled Tonebender Mark 1.5 inspired by Chase Bliss's Preamp Mark II, with moving faders, preset recall, and MIDI. Without compromising on its analog heritage.

## Goals:
- [ ] Tonebender Mark 1.5 PNP Germanium
- [ ] Moving Faders + Fader control
- [ ] MIDI Control / Preset Recall
- [ ] Relay system
- [ ] Soft-switching system

## BOM
Bom to be updated when schematics and PCBs finished

## Boards
- Control board:
  - The board with:
    - Pi Pico
    - 7 Segment display
    - Motor drivers
    - 9v -> 5v -> 3.3v power supply
- Fader board
  - Board dedicated to faders
- Eyelet board
  - The eyeletboard constructed PNP germanium fuzz face
